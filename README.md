## Full Featureset

### - The Primordial Toolset

A full arsenal of **Primordial** tools. A Hard Hammer, Soft Mallet, Wrench, Crowbar, Screwdriver, Wire Cutters, Saw, File, Knife, Mortar, Plunger, and a Healing Axe. Each can be toggled on or off individually in the config yaml

### - Dual Mining Modes

Crouch + Scroll to swap any Primordial tool between **Insta Mine** and **Precision Mining** (configurable speed). 

### - Relocate Mined Blocks

Drops automatically sent to your inventory (configurable)

### - Animated Primordial Shader

Primordial tools render with an animated shader inspired by Avaritia's "Sword of the Cosmos"! Purely visual and can be disabled in the config for lower-end machines.

### - Tool Sound Rework

Reworked in-hand use sounds, bringing back the OG GT:NH wrench sounds! 

### - Configuration

A YAML config lets you toggle each tool, tune precision mining speed, switch block relocation, and enable or disable the shader. More options to come, suggestions welcome!

## Installation

Drop `gtsf-<version>.jar` into `mods/`.

Required: [GregTech CEu: Modern](https://www.curseforge.com/minecraft/mc-mods/gtceu-modern) (7.5.3).
