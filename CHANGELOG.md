# Changelog

All notable changes to **GregTech: Star Foundry** are documented here.

## [0.1.1] - 2026-07-05

### added
- In-game config screen with labelled **General** and **Items** categories.
- `Precision Mining Speed` config option to tune Precision Mining (default: 8).
- `Primordial Shader` config toggle to enable or disable the animated tool shader.

### changed
- Restructured the config into grouped sections.
- Polished the Primordial tool rendering and cosmic shader model.
- Fixed the Healing Axe and Primordial tool behavior.

## [0.1.0] - 2026-07-01

### added
- Initial release!
- The Primordial toolset at the new **Godforged** harvest tier: Hard Hammer, Soft
  Mallet, Wrench, Crowbar, Screwdriver, Wire Cutters, Saw, File, Knife, Mortar,
  Plunger, and Healing Axe.
- Dual mining modes (Instant Mining / Precision Mining) with Crouch + Scroll switching.
- Send Mined Blocks straight to the inventory.
- Primordial cosmic shader on tools.
- Tool sound rework for GregTech tools (wrench use sounds).
- Per-tool enable/disable configuration.
